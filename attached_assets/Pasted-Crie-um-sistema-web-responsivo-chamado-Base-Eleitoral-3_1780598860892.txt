Crie um sistema web responsivo chamado “Base Eleitoral 360”.

Objetivo:
Criar um dashboard/app moderno para monitoramento territorial de campanha eleitoral, com foco em candidato, coordenação, lideranças, apoiadores, prospecção de eleitores, zonas eleitorais, mapas e comparativos estratégicos.

Estética:
Interface moderna, leve, profissional e intuitiva.
Visual inspirado em dashboards de tecnologia, apps de mobilidade, delivery e inteligência de dados.
Usar cards, ícones, gráficos, menu lateral, filtros e layout limpo.
Paleta sugerida:
- Azul escuro para gestão e confiança
- Verde para crescimento e áreas fortes
- Amarelo para atenção
- Vermelho para áreas críticas
- Roxo/azul neon para tecnologia
- Fundo claro com elementos em branco e cinza suave

Stack:
Use React com Vite ou Next.js, TailwindCSS e componentes modernos.
Caso seja mais simples para o Replit, pode começar com React + Vite + Tailwind.
Criar inicialmente com dados mockados/fictícios, mas preparar a estrutura para depois conectar banco de dados.

Estrutura do menu lateral:
1. Dashboard Geral
2. Mapa de Força
3. Lideranças
4. Apoiadores / Pessoas
5. Prospecção
6. Mapa RJ
7. Mapa Maricá
8. Zonas Eleitorais
9. Comparativo Eleitoral
10. Agenda de Campo
11. Demandas
12. Relatórios
13. Configurações
